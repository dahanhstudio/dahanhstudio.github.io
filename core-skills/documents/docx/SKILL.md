# Kỹ năng: Xử lý DOCX

Kỹ năng này xử lý các tương tác với tài liệu Microsoft Word (.docx).

## Khả năng
- Tạo file .docx mới từ văn bản thô.
- Chỉnh sửa tài liệu hiện có (thêm văn bản, thay thế nội dung).
- Trích xuất văn bản và định dạng từ tài liệu.
- Xử lý các thay đổi được theo dõi (track changes) và bình luận.

## Hướng dẫn cho AI (IDE)
Khi người dùng cần tạo báo cáo, đơn từ, hoặc bài luận dưới dạng Word:
1.  **Format**: Sử dụng định dạng Markdown làm trung gian nếu cần, sau đó chuyển đổi sang cấu trúc tài liệu.
2.  **Chính xác**: Giữ nguyên định dạng của các tiêu đề (Heading 1, 2, 3).
3.  **Thư viện**: Nếu cần viết script, ưu tiên thư viện `python-docx` (Python) hoặc thư viện tương đương trong Node.js.
