# Kỹ năng Mẫu (Template Skill)

Đây là mẫu chuẩn để tạo các kỹ năng mới cho Agent.

## Cấu trúc
Một kỹ năng bao gồm một thư mục chứa file `SKILL.md`.

## Frontmatter (Phần đầu file)
File bắt buộc phải bắt đầu bằng YAML frontmatter:
- `name`: Định danh duy nhất (dùng kebab-case, chữ thường gạch nối).
- `description`: Mô tả ngắn gọn kỹ năng làm gì.

## Hướng dẫn cho AI (IDE)
Viết các hướng dẫn chi tiết tại đây để Agent đọc và làm theo khi kích hoạt kỹ năng này.

## Ví dụ
- Cung cấp các ví dụ rõ ràng về cách sử dụng kỹ năng.

## Nguyên tắc
- Liệt kê các ràng buộc hoặc các thực hành tốt nhất (best practices).
