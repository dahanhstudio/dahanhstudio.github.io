# Kỹ năng: Xây dựng Artifact (Artifacts Builder)

Kỹ năng này chuyên tạo ra các file HTML chất lượng cao, có tính tương tác.

## Khả năng
- Tạo ứng dụng React hoàn chỉnh chỉ trong một file HTML duy nhất.
- Sử dụng Tailwind CSS để tạo kiểu nhanh và đáp ứng (responsive).
- Tích hợp các components kiểu shadcn/ui để có giao diện đẹp.

## Hướng dẫn cho AI (IDE)
Khi tạo Dashboard, Công cụ tính toán, hoặc Giao diện tương tác:
1.  **Single File**: Đảm bảo toàn bộ HTML, CSS, và JS nằm trong một file duy nhất để dễ chia sẻ.
2.  **CDN**: Sử dụng React, ReactDOM, và Babel qua CDN.
3.  **Icons**: Sử dụng `lucide-react` cho các biểu tượng.
4.  **Thẩm mỹ**: Luôn đảm bảo sản phẩm cuối cùng nhìn chuyên nghiệp và hoạt động mượt mà.
