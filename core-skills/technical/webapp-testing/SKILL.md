# Kỹ năng: Kiểm thử WebApp (WebApp Testing)

Kỹ năng này kích hoạt khả năng kiểm thử tự động cho ứng dụng web.

## Khả năng
- Viết kịch bản kiểm thử Playwright.
- Thực thi kiểm thử UI (click, gõ phím, điều hướng).
- Xác minh các phần tử trang và trạng thái.
- Gỡ lỗi UI bằng cách chạy test và phân tích kết quả.

## Hướng dẫn cho AI (IDE)
Khi người dùng muốn kiểm tra lỗi, verify tính năng web, hoặc tái hiện bug:
1.  **Kịch bản**: Xác định rõ các bước người dùng thực hiện (User Flow).
2.  **Assertions**: Luôn bao gồm các bước kiểm tra (assert) để xác nhận kết quả đúng/sai.
3.  **Playwright**: Sử dụng Playwright làm công cụ chính.
4.  **Môi trường**: Giả định chạy trong môi trường browser headless trừ khi được yêu cầu khác.
