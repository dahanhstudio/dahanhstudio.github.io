# Kỹ năng: Nghệ thuật Thuật toán (Algorithmic Art)

Kỹ năng này cho phép Agent tạo ra các tác phẩm nghệ thuật generative bằng thư viện p5.js.

## Khả năng
- Tạo mã nguồn p5.js để vẽ hình.
- Sử dụng hàm ngẫu nhiên có hạt giống (seeded randomness) để đảm bảo kết quả có thể tái tạo.
- Triển khai các thuật toán phức tạp như flow fields (trường vector) và particle systems (hệ thống hạt).
- Tạo các hoa văn thị giác và nghệ thuật trừu tượng.

## Hướng dẫn cho AI (IDE)
Khi người dùng yêu cầu tạo "nghệ thuật", "tranh code", hoặc "generative art":
1.  **Code**: Viết mã p5.js nhúng trong một file HTML duy nhất (Artifact).
2.  **Thư viện**: Sử dụng CDN tiêu chuẩn cho p5.js.
3.  **Thẩm mỹ**: Tập trung vào hiệu ứng thị giác đẹp mắt, màu sắc hài hòa.
4.  **Tương tác**: Nếu có thể, hãy thêm tương tác chuột hoặc bàn phím đơn giản.

## Nguyên tắc
- Code phải tự chạy được (self-contained).
- Tránh các vòng lặp vô hạn gây treo trình duyệt.
