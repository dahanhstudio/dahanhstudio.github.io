# Kho Kỹ Năng Agent (Agent Skills Repository)

Kho lưu trữ này chứa một tập hợp các kỹ năng (skills) để mở rộng khả năng của AI Agent.

## Cấu trúc

- **template/**: Mẫu cơ bản để tạo kỹ năng mới.
- **creative/**: Kỹ năng về thiết kế, nghệ thuật và giao diện.
  - `algorithmic-art`: Nghệ thuật thuật toán với p5.js.
  - `frontend-design`: Hỗ trợ thiết kế UI/UX.
  - `theme-factory`: Tạo theme cho các sản phẩm.
- **documents/**: Kỹ năng xử lý tài liệu văn phòng.
  - `docx`: Xử lý văn bản Word.
  - `pdf`: Tạo và trích xuất PDF.
  - `pptx`: Tạo bài thuyết trình PowerPoint.
- **technical/**: Kỹ năng phát triển và kiểm thử.
  - `mcp-builder`: Hướng dẫn xây dựng MCP server.
  - `webapp-testing`: Kiểm thử ứng dụng web với Playwright.
  - `artifacts-builder`: Xây dựng các tạo tác HTML phức tạp.
- **enterprise/**: Kỹ năng trong bối cảnh doanh nghiệp.
  - `brand-guidelines`: Áp dụng bộ nhận diện thương hiệu.
  - `internal-comms`: Viết truyền thông nội bộ.
- **reverse-engineering/**: Quy trình dịch ngược và tái cấu trúc (Reverse Engineering).
  - `diagnosis`: Chẩn đoán và phân tích (Phase 0).
  - `mapping`: Lập bản đồ chiến lược (Phase 1).
  - `deconstruction`: Phân rã code cũ (Phase 2).
  - `reconstruction`: Tái cấu trúc hệ thống (Phase 3).
  - `verification`: Xác minh và bàn giao (Phase 4).
